🌐 Landing Page

Uma Landing Page estática desenvolvida com HTML, focada em apresentação de conteúdo de forma clara, objetiva e responsiva.

🚀 Sobre o projeto

Este projeto consiste em uma página única (single page) criada para apresentar um produto, serviço ou portfólio.

O foco está em:

Design limpo e moderno Carregamento rápido ⚡ Estrutura simples e eficiente Facilidade de customização 🛠️ Tecnologias utilizadas HTML5 CSS3 (inline ou arquivo separado, se aplicável) (Opcional) JavaScript básico 📂 Estrutura do projeto landing-page/ ├── index.html ├── assets/ │ ├── css/ │ ├── img/ │ └── js/

⚠️ Caso seu projeto tenha apenas o index.html, você pode simplificar ainda mais:

landing-page/ └── index.html ⚙️ Como rodar o projeto

Como é uma página estática, você não precisa instalar nada.

Opção 1: Abrir diretamente

Basta abrir o arquivo:

index.html

ou dar duplo clique nele.

Opção 2: Usar servidor local (recomendado)

Se quiser rodar como ambiente web:

Com VS Code (Live Server) Instale a extensão Live Server Clique com o botão direito no index.html Clique em Open with Live Server Com Node.js npx serve . ✨ Funcionalidades 📱 Layout responsivo 🎯 Seções organizadas (ex: hero, sobre, serviços, contato) ⚡ Carregamento rápido 🧩 Estrutura simples para fácil edição 🎯 Possíveis usos Página de produto Página de serviço Portfólio pessoal Página de captura (leads) Apresentação institucional 🔧 Customização

Para adaptar o projeto:

Edite os textos diretamente no index.html Substitua imagens na pasta assets/img
